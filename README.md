# iOS(16.4以上)の人  
1. 「**Safariで**」リンクを開く
2. 共有マークから [ホーム画面に追加] を選択
3. ホーム画面から開いて最初に通知許可をする

# Androidの人
1. Chromeで開く
2. ︙マーク -> [ホーム画面に追加] -> [インストール]
3. 最初に通知許可で行けるはず(動作未確認)
